# prak-3-kurs-hw2
```
1.substring: на вход подается любая строка, выводится число повторений t и подстрока
2.mostpopularword: на вход подается любой текст, выводится самое популярное слово или -, если их несколько
3.phone: на вход подается любое число и выводитвсе возможные комбинации букв(1 заменена на _)
4.graph_search: на вход подаются ребра записанные в виде : [[r1,r2],...,[rn,rn]] и начальная вершина, выводятся посещенные вершины во время обхода в ширину и в длину
5.path_search: на вход подаются ребра записанные в виде : [[r1,r2,w1],...,[rn,rn,wn]], начальная и конечная вершины, выдается кратчайший путь
6.time_delay: на вход подаются ребра записанные в виде : [[r1,r2,w1],...,[rn,rn,wn]] и узел, выдается наименьшее время задержки
в 4,5,6 нумерация вершин с 0
```
